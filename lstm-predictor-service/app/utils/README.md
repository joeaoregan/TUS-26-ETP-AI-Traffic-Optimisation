# Utilities Module Documentation

## metrics.py

The `metrics.py` module provides functions and classes for evaluating the performance of various models. It includes methods for calculating common evaluation metrics such as accuracy, precision, recall, F1 score, and more. This module is essential for assessing the effectiveness of models used in traffic optimization.

## feature_engineering.py

The `feature_engineering.py` module includes functions that transform raw data into meaningful features that can improve model performance. This may involve scaling, encoding categorical variables, handling missing values, and creating interaction terms. Proper feature engineering is critical in enhancing the predictive capability of machine learning models.

## stationarity.py

The `stationarity.py` module is designed to assess whether a time series is stationary or non-stationary. It implements tests like the Augmented Dickey-Fuller test and offers tools for differencing and transforming series to achieve stationarity. Ensuring stationarity is key for many time series forecasting models to perform accurately.
