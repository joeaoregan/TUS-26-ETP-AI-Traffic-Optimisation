# Trained Models

This directory stores saved LSTM models and associated artifacts.

## Contents

- `lstm_model.h5` - Trained LSTM model weights (TensorFlow/Keras format)
- `scaler.pkl` - MinMaxScaler for data normalization
- `model_config.json` - Model architecture and hyperparameters

## Usage

Models are loaded by the inference service to make traffic predictions.

## Training

Models are generated by running the training script:
```bash
python train_lstm.py
