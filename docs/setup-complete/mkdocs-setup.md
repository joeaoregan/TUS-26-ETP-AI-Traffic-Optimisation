# MkDocs Setup

Install mkdocs:

```bash
pip install mkdocs
```

To build MkDocs:

```bash
python -m mkdocs build
```

To run documenation locally <http://127.0.0.1:8000/>:

```bash
python -m mkdocs serve 
```

To deploy documentation to GitHub pages, after updating (Commit first).

```bash
python -m mkdocs gh-deploy
```
